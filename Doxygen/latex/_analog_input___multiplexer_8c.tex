\subsection{Detailed Description}
The analog sampling circuitry of the Tekdaqc can only sample a single channel at a time and so it needs to multiplex the various analog input channels to the A\-D\-C. The Tekdaqc incorporates both an external analog input multiplexer and the A\-D\-C's internal input multiplexer, and the methods in this file will ensure that both the external multiplexer and the A\-D\-C's internal multiplexer are properly configured.
