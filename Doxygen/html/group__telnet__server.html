 Data structure to hold the state of the Telnet server. Contains all of the necessary state variables to impliment the Telnet server. Direct manipulation of these members is not recommended as it may leave the server in an inconsistent state. Instead, helper methods are provided which will ensure that all necessary operations occur as a result of any change. More...

enum TelnetState_t
Enumerator
STATE_NORMAL 

The telnet option parser is in its normal mode. Characters are passed as is until an IAC byte is received.

STATE_IAC 

The previous character received by the telnet option parser was an IAC byte.

STATE_WILL 

The previous character sequence received by the telnet option parser was IAC WILL.

STATE_WONT

void TelnetProcessDo ( char  option)

This function will handle a DO request for a telnet option. If it is an option that is known by the telnet server, a WILL response will be generated if the option is not already enabled. For unknown options, a WONT response will always be generated.

The response (if any) is written into the telnet transmit buffer.
